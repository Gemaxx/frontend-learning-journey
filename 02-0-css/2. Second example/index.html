Display property

inline-block Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height, width, padding, and margin values
Display: none The element is completely hidden from the document flow (does not take up any space).
Visability: hidden The element will be hidden, (but it will take up a space)
